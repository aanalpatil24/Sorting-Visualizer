# Sorting Algorithm Visualizer

An **interactive web-based tool** to visualize and understand how popular sorting algorithms work step-by-step through smooth, real-time animations.  
Perfect for students, educators, and anyone curious about how sorting actually happens under the hood.

---

## Features
<details>
  <Summary> Click on the Feauters</Summary>
- Multiple Algorithms Supported
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  

- Adjustable Controls
  - Change **array size** dynamically  
  - Control **animation speed** in real time.

- Random Array Generation
  - Instantly generate new unsorted arrays to test different cenarios. 

- Responsive & Intuitive UI
  - Works seamlessly on desktop and mobile.
  - Visual color cues for comparisons, swaps, and sorted sections.

- Educational & Engagin
  - Perfect for learning sorting logic through live animation.
</details>
