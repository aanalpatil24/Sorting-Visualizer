# Sorting Algorithm Visualizer

An **interactive web-based tool** to visualize and understand how popular sorting algorithms work step-by-step through smooth, real-time animations. Perfect for students, educators, and anyone curious about how sorting actually happens under the hood.

---

## Features
<details>
  <Summary> Click on the Feauters</Summary>
- Multiple Algorithms Supported
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  

- Adjustable Controls
  - Change **array size** dynamically  
  - Control **animation speed** in real time.

- Random Array Generation
  - Instantly generate new unsorted arrays to test different scenarios. 

- Responsive & Intuitive UI
  - Works seamlessly on desktop and mobile.
  - Visual colour cues for comparisons, swaps, and sorted sections.

- Educational & Engaging
  - Perfect for learning sorting logic through live animation.
</details>
---

## Tech Stack

- **Frontend:** HTML5, CSS3 / Tailwind CSS  
- **Logic & Animation:** JavaScript (Vanilla JS or React)  
- **Version Control:** Git/Github
